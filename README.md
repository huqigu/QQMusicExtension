# QQMusicExtension

### 说明
MacOS QQ音乐客户端插件，VIP歌曲可试听1分钟。

### 安装
```
cd QQMusicExtension
./Rely/Install.sh
```

### 卸载
```
cd QQMusicExtension
./Rely/Uninstall.sh
```
